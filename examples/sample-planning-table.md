# Sample Sticker Planning Table

This is an example planning table for a 16-sticker set featuring a cute military character (Q版小兵).

## Character Concept

- **Name**: 小兵阿勇
- **Style**: Chibi/kawaii, thick black outlines, flat illustration
- **Outfit**: Green camouflage uniform, military helmet with Taiwan ROC flag
- **Features**: Round face, big expressive eyes, rosy cheeks

## Sticker Plan

|---|------|--------|---------|----------|

## Category Balance Check

- ✅ Greetings: #10 (早安), #15 (晚安)
- ✅ Affirmatives: #01 (收到), #02 (讚), #06 (是長官), #13 (OK)
